honeygain-闲置设备挂机和IP代理平台

使用Honeygain，您只需分享您的互联网即可赚钱。只支持家庭宽带和流量挂机

注意！家庭宽带使用IP类挂机软件可能导致你的IP在https://www.ip2location.com/ 被标记为住宅代理，可能出现频繁输入验证码的情况，请谨慎使用！

由于honeygain挂机api使用cloudflare，2022年3月开始联通宽带出现连接问题，国内挂机可能出现频繁掉线收益过少的问题

介绍

引用官网介绍

使用Honeygain，您只需分享您的互联网即可赚钱。

工作原理

官网上说软件会利用其用户未使用的数据网络来帮助他自己的客户完成智能网络计算、市场调研和内容接收等服务。

注册过程

使用邀请链接注册可以获得5美元，

Honeygain注册https://good.zeroyuki.com/jump/4.html

挂机规则

不允许挂VPN来分享流量

不允许恶意多开，包括：沙盒、强制多开等

尽量不要使用vps、虚拟机、模拟器等

同一个局域网、公网IP下不允许出现多于2个的Honeygain设备，第三个Honeygain设备将报错不分享流量

不支持IPv6

不允许服务器网络，阿里云、腾讯云等主机商的服务器网络

挂机教程

挂机软件下载https://www.honeygain.com/download/

windows电脑，安卓手机，mac电脑，ios手机下载软件选择 accept 然后输入账号登陆即可

linux服务器需要使用docker

docker安装命令

curl -fsSL https://get.docker.com | bash -s docker

然后执行

docker run honeygain/honeygain -tou-get

docker run honeygain/honeygain -tou-accept -email 你注册的邮箱 -pass 你账号的密码 -device 设备名称

积分制度与提现渠道

Honeygain分为两种积分，当日积分，即今天一天（24小时）所得的积分；总积分，即总账户积分，用于提款1000积分为 1美元。流量遵照 10MB 为 1积分 的原则增长当天积分。积分会在早上8点会自动换算时自动添加到总积分。

每日的8点以后有Honeygain签到，每日签到获得积分最高为10000积分，你可以靠签到获取积分。

攒到3000积分可以提现，Honeygain支持两种提现方式比特币或PayPal。
